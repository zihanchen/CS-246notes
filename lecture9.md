##Pointers and Arrays  

#####Example  
```C++
#include <iostream>
using std::cout;
using std::endl;
int main() {
	int i = 10;

	int *ptr;

	ptr = &i;
	*ptr = 2;

	int a[4] = {1, 2, 3, 4};

	ptr = a;

	cout << a[i] << endl; //prints 3 ; *((a)+(i))
	cout << ptr[i] << endl; //also prints 3

	cout << 5["abcdef"] << endl; // prints f
	cout << 2[ptr] << endl; // prints 3
}
```
